# AI, RAG & Agents — Sessions Overview

12 sessions | 1.5–3h each | Free tools only (Ollama, LangChain, Chroma, HuggingFace)

---

**Session 1 (1.5–2h): AI & LLM Foundations + AI Profession Landscape**
🛠 Build: CLI prompt-based assistant answering domain questions

**Session 2 (2h): Prompt Engineering & Evaluation**
🛠 Build: Prompt test harness — 10 benchmark prompts with scoring sheet

**Session 3 (2h): Embeddings & Vector Databases**
🛠 Build: Semantic search app over a set of public documents

**Session 4 (2–2.5h): RAG Fundamentals**
🛠 Build: End-to-end local RAG chatbot over a public dataset

**Session 5 (2h): Advanced RAG & Graph RAG**
🛠 Build: RAG with metadata filtering, citation display, and agentic retrieval

**Session 6 (2h): LangChain — Orchestrating LLM Apps**
🛠 Build: Multi-step pipeline (summarize → classify → recommend)

**Session 7 (2–3h): AI Agents Fundamentals**
🛠 Build: Tool-using agent (web search / file reader / calculator, free APIs)

**Session 8 (2–3h): LangGraph — Multi-Agent Patterns**
🛠 Build: Two-agent workflow (researcher + writer) with reflection loop

**Session 9 (2h): MCP — Model Context Protocol**
🛠 Build: Custom MCP server + client that exposes and calls one local tool

**Session 10 (2h): Safety, Guardrails & Production Readiness**
🛠 Build: Agent pipeline with output validation, latency tracking, and fallback model

**Session 11 (2–3h): Capstone Build Sprint** *(team project)*
🛠 Build: Full AI app of your choice — RAG assistant, agent, or MCP tool hub

**Session 12 (1.5–2h): Demo Day & Career Readiness**
🎤 Demo: Present your project — architecture, trade-offs, and CV-ready bullets
